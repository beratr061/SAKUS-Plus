# Android Kotlin Development

Bu projede Kotlin/Android dosyaları üzerinde çalışırken **android-kotlin-development** skill'ini uygula.

- MVVM mimarisi ve Jetpack ViewModel kullan
- Jetpack Compose ile UI oluştur
- Retrofit ile API çağrıları, Room ile yerel depolama
- Hilt ile dependency injection
- StateFlow ve coroutines ile asenkron işlemler
